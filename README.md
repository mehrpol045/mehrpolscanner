## ✨ ویژگی‌ها | Features

| ویژگی | Feature | توضیح | Description |
|-------|---------|-------|-------------|
| 🔍 اسکن IP | IP Scan | کشف سریع IP های سالم Cloudflare | Fast discovery of clean Cloudflare IPs |
| 📡 DNS Hunter | DNS Hunter | مقایسه پاسخ DNS با baseline معتبر | Compare DNS responses with a valid baseline |
| 🌐 بررسی دامنه | Domain Check | چک دسترسی به سایت‌های مختلف | Check access to various websites |
| 🔒 SNI Check | SNI Check | بررسی SNI و تشخیص فیلترینگ | Detect SNI-based filtering |
| 📊 Diagnostics | Diagnostics | تشخیص مشکلات DNS، routing، TCP | Diagnose DNS, routing, TCP issues |
| 📈 Monitor | Monitor | مانیتورینگ پس‌زمینه‌ای IP های ذخیره شده | Background monitoring of saved IPs |
| 📤 Export | Export | خروجی V2Ray، Xray، Clash، CSV | Export to V2Ray, Xray, Clash, CSV |

## 📥 روش استفاده | How to Use

**فارسی:**
1. APK را از بخش [Releases](https://github.com/mehrpol045/mehrpolscanner/releases) دانلود و نصب کنید
2. برنامه را باز کرده و روی **اسکن** بزنید
3. IP مناسب را انتخاب و با **Export** کانفیگ بسازید
4. کانفیگ را به برنامه VPN وارد کنید

**English:**
1. Download and install the APK from [Releases](https://github.com/mehrpol045/mehrpolscanner/releases)
2. Open the app and tap **Scan**
3. Select a suitable IP and generate config with **Export**
4. Import the config into your VPN app
