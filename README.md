## ✨ ویژگی‌ها | Features

- 🔍 **اسکن IP** — کشف سریع IP های سالم Cloudflare
- 📡 **DNS Hunter** — مقایسه پاسخ DNS ایرانی با baseline معتبر
- 🌐 **بررسی دامنه** — چک دسترسی به سایت‌های مختلف
- 🔒 **SNI Check** — بررسی SNI و تشخیص فیلتر
- 📊 **Diagnostics** — تشخیص مشکلات DNS، routing، TCP
- 📈 **Monitor** — مانیتورینگ پس‌زمینه IP های ذخیره شده
- 📤 **Export** — خروجی V2Ray، Xray، Clash، CSV
